ECAD-ZIP-To-Index.PHP v0.1.00


How it works:
1. zip your files
2. rename the created zip file to index.zip
3. put the zip file into the same directory as install_creator.php
4. execute / open install_creator.php in your browser / or localy
5. a new file with the name index.php will be created in the same directory

6. when the newly created file index.php is executed it will decompress itself and refresh the page
